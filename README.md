# Chatterly

A secure, real-time chat app with a beautiful UI and fast messaging experience using WebSocket technology.

## Features
- 1-on-1 real-time messaging
- Authentication
- Message history

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Socket.IO

## Getting Started

```bash
git clone https://github.com/Ronel1409/chatterly.git
cd chatterly
npm install
npm start
